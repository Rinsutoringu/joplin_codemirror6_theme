# Changelog

## [0.2] - 2025-12-03

### Added
- Customizable inline code styling with background and text color
- Link and URL color customization
- GitHub Alerts support (NOTE, TIP, IMPORTANT, WARNING, CAUTION)
- Enhanced heading styles for H1-H6
- Blockquote styling improvements
- Keyboard shortcuts for heading levels (Ctrl+1 through Ctrl+6)
- Comprehensive settings panel with toggles for all features
- Color pickers for inline code and link colors

### Features
- All styling features can be toggled on/off individually
- Settings persist across Joplin sessions
- Real-time style updates when toggling features
- Intelligent heading toggle - press once to add, press again to remove
- Multi-line heading support

### Technical
- Built on CodeMirror 6 extension system
- Uses Joplin plugin API for settings management
- Markdown-it plugin integration for preview rendering
